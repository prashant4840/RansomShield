# UI Wireframe (SOC Dashboard)

## Screen Layout
1. **Top Header**
   - Product title: `RansomShield SOC`
   - Subtitle: `AI-Powered Early Warning for Android`

2. **Control Card**
   - Toggle: Live Monitoring
   - Toggle: Threat Simulation Mode
   - Gamified Security Score

3. **Threat Timeline Card**
   - Real-time line chart of risk score samples

4. **Current Threat Card**
   - Risk level badge (Low/Medium/High)
   - Numeric score
   - Explainable reasons

5. **AI Assistant Card**
   - Plain-language recommended actions

6. **One-Tap Action Card**
   - Kill process
   - Lock folders
   - Rollback
   - Safe mode

7. **App Risk Ranking List**
   - Ranked suspicious apps with package and score

## Design Language
- Dark background gradient
- Neon cyan / teal accent highlights
- Semi-transparent glass cards
- Minimal, SOC-inspired spacing and typography
